# Components

This folder contains all component documentation files.

Each `.json` file represents one component from the ABC design system. The workspace tool reads these files and renders them dynamically.

To add a new component:
1. Generate the component JSON in a Claude.ai conversation with the Figma MCP connected
2. Upload the file here (or create it directly via "Add file → Create new file")
3. Commit the change
4. Refresh the live workspace — the new component will appear in the sidebar
